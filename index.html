The Future of Notetaking

  • Virtual Reality Note Taking System is a innovative project that we made at HackingEDU in San Mateo, California.
  • This is aimed towards revolutionizing education. We are using an Oculus Rift DK2 and Leap motion sensor to make notes in 3D space, and further saving the environment to view notes later in the 3D environment
  • The tracking is possible using Leap motion and the notes are display inside the Oculus. The user can take notes in 3D with simply writing in air with their fingertips, using skeletal structure detection.
  • Project: Devpost
  • Date: October, 2015
  • Category: Hackathon

HackZurich

  • This project, CoverMe, was made at a computer science conference in Zurich, Switzerland. I had the opportunity to work with talented PhD students from University of Munich and full time software engineers.
  • CoverMe is a multi platform application to formulate insurance prices, automating insurance rates based on numerous visual recognition API's (Google Visual, Microsoft Oxford, IBM Watson).
  • The user needs to simply take a picture of the item they need to insure, and sign in with either Facebook, linkedIn or Google+ (Using those API's as well to send us user information for generating health insurance rates) and then processing algorithm lists the information to review.
  • Finally the user can then instantly buy the insurance rate from the app (PayPal API)
  • Project: Github
  • Date: September, 2016
  • Category: Hackathon

Recruitbot, Yale University

At YHacks, a competition at Yale University, our group developed a smart software system geared towards recruiters, large companies and universities. Our project, RecruitBot, takes in any number of resumes as input, and lets the user rank these documents. The user can set criteria and their corresponding weight. The user can choose from criteria such as GPA, university ranking, prior work experience, technical skills known, and much more! We were pleased to be named one of the winners of this conference, and the winner of the Best Recruiting App award.

  • Project: Devpost
  • Date: November, 2015
  • Category: Hackathon
